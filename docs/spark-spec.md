# Spark — Продуктовая спецификация

**Версия:** 1.4 (PRD)  
**Дата:** Март 2026

---

## 1. Обзор продукта

**Spark** — это персональный инструмент для изучения иностранных языков с помощью видеоконтента с YouTube. Основная идея продукта заключается в "сегментации": пользователь не просто смотрит видео целиком, а разбивает его на важные учебные отрезки (сегменты).

Каждый сегмент имеет четкие границы по времени и текстовое описание (например, перевод фразы или пояснение грамматики). Приложение позволяет объединять такие сегменты в тематические коллекции ("сегментированные видео") и изучать их последовательно — плеер автоматически переходит от одного учебного фрагмента к другому.

### Целевая аудитория
Люди, самостоятельно изучающие иностранные языки, которые хотят структурировать учебный материал из YouTube и фокусироваться на конкретных фразах или темах.

### Платформы
Приложение работает в браузере и полностью адаптировано для двух режимов использования:
*   **Мобильная версия**: Оптимизирована для управления одной рукой «на ходу».
*   **Десктопная версия**: Использует преимущества широких экранов для комфортной работы и редактирования контента.

## 2. Общие принципы интерфейса

### Адаптивность и навигация
Приложение автоматически подстраивает интерфейс под размер экрана:

1.  **На мобильных устройствах**:
    *   Основная навигация расположена в нижней части экрана (Таббар) для удобного доступа большим пальцем.
    *   Доступны вкладки «Библиотека видео» и «Добавить контент».
    *   Каждая страница имеет заголовок с кнопками быстрых действий и переключателем темы оформления.

2.  **На больших экранах (ПК и планшеты)**:
    *   Навигация перемещается в фиксированную боковую панель слева. Панель содержит логотип и пункты меню с текстовыми подписями.
    *   Верхняя шапка скрывается, а элементы управления и заголовки интегрируются непосредственно в рабочую область страницы.
    *   Кнопки действий становятся более крупными и содержат поясняющий текст вместо одиночных иконок.

### Темы оформления
Приложение поддерживает светлый и темный режимы. Пользователь может переключать их вручную, выбор сохраняется и применяется автоматически при следующих визитах.

## 3. Экран: Библиотека видео (SegmentedVideosPage)

Главный экран приложения, где хранятся все созданные пользователем коллекции.

### Функционал
*   **Полки с видео**: Каждая тематическая коллекция представлена в виде «полки» (горизонтального ряда). Слева отображается название коллекции, справа — общее количество учебных фрагментов в ней.
*   **Прокрутка**: На мобильных устройствах полки листаются привычным жестом свайпа. На ПК при наведении на полку появляются стрелки навигации, позволяющие пролистывать карточки страницами.
*   **Карточки фрагментов**: Каждый фрагмент в ряду имеет превью и ведет на экран плеера. На больших экранах размер карточек увеличивается для лучшей читаемости.
*   **Быстрый переход**: Название коллекции кликабельно и ведет к полному списку её сегментов.

### Состояния
*   **Пустая библиотека**: Если пользователь еще ничего не создал, отображается приветственный экран с описанием преимуществ и кнопкой для создания первого видео.

---

## 4. Экран: Коллекция сегментов (SegmentedVideoPage)

Страница детального обзора конкретной коллекции видеофрагментов.

### Функционал
*   **Список фрагментов**: Вертикальный список всех сегментов с превью, описанием и четким указанием временных границ (например, "0:30 – 1:45").
*   **Изменение порядка**: Пользователь может свободно менять порядок фрагментов в списке простым перетаскиванием (Drag & Drop). На компьютерах элементы снабжены специальной иконкой захвата.
*   **Запуск обучения**: Крупная кнопка «Воспроизвести» запускает последовательный просмотр всех фрагментов коллекции, начиная с первого.
*   **Управление**: Возможность добавить новый фрагмент в эту коллекцию или полностью удалить саму коллекцию через меню действий.

---

## 5. Экран: Учебный плеер (SegmentPage)

Ключевой экран, где происходит процесс обучения.

### Режимы отображения
1.  **Мобильный вид**: Видеоплеер закреплен в верхней части экрана, описание и настройки контента располагаются под ним и прокручиваются вертикально.
2.  **Десктопный вид**: Экран разделен на две части. Основную область занимает плеер и описание. Справа появляется боковая панель со списком всех фрагментов данной коллекции для мгновенного переключения между ними.

### Возможности плеера
*   **Умное воспроизведение**: Плеер работает только в границах заданного сегмента. По достижении конца фрагмента приложение автоматически переходит к следующему.
*   **Зацикливание**: Пользователь может включить режим повтора. В этом случае после последнего фрагмента коллекции воспроизведение начнется заново с первого.
*   **Кастомное управление**:
    *   Кнопки перемотки и паузы.
    *   Изменение скорости видео (от замедленного 0.5x до ускоренного 2x) для разбора быстрой речи.
    *   Полноэкранный режим.
    *   Регулировка громкости и отключение звука.
*   **Автоматизация**: Контроллеры плеера автоматически скрываются при просмотре, чтобы не отвлекать от контента, и появляются при активности пользователя.

## 6. Создание и редактирование контента

### Процесс добавления
Добавление нового материала организовано максимально просто:
1.  **Ссылка**: Пользователь вставляет ссылку на YouTube-видео.
2.  **Предпросмотр**: Приложение мгновенно подгружает видео для проверки.
3.  **Выбор интервала**: С помощью удобного слайдера с двумя ползунками прямо под видео пользователь отмечает начало и конец учебного фрагмента. Также доступен ручной ввод времени до секунды.
4.  **Описание**: Добавление текста (перевод, примечания), который будет виден при просмотре.

### Валидация
*   Приложение проверяет доступность видео на YouTube перед сохранением.
*   Если видео недоступно для встраивания или ссылка неверна, пользователь получит понятное текстовое уведомление.
*   Конец фрагмента не может быть раньше его начала.

## 7. Техническая основа продукта

### Хранение данных
*   Все коллекции и настройки пользователя хранятся локально в браузере. Это обеспечивает мгновенный доступ к данным при повторном открытии страницы без необходимости регистрации (на текущем этапе).
*   Изменения (например, перетаскивание элементов) сохраняются автоматически с небольшой задержкой, чтобы не замедлять работу интерфейса.

### Ограничения текущей версии
*   Данные привязаны к конкретному браузеру и устройству.
*   Автоматическое воспроизведение видео может начинаться без звука в соответствии с политикой современных браузеров (требуется первое действие пользователя для включения звука).

## 8. Планы по развитию

*   **Синхронизация**: Возможность доступа к своей библиотеке с разных устройств через облачный аккаунт.
*   **Интерактивные упражнения**: Режим тренировок (флэш-карточки) на основе описаний к сегментам.
*   **Импорт/Экспорт**: Возможность делиться своими учебными коллекциями с другими пользователями.
